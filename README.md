Guia do Projeto – Currículo em Blazor
Guia de Execução

Para executar o projeto, siga os passos abaixo utilizando o terminal:

Primeiro, acesse a pasta do projeto:
cd MeuCurriculo

Em seguida, restaure as dependências necessárias:
dotnet restore

Depois, execute o projeto:
dotnet run

Após a execução, o sistema será iniciado automaticamente. O terminal exibirá um endereço (URL), geralmente no formato:
https://localhost:xxxx

Basta copiar esse endereço e colar no navegador para visualizar a aplicação.

Tecnologia Utilizada

O projeto foi desenvolvido utilizando a tecnologia Blazor WebAssembly, que faz parte da plataforma .NET. Essa tecnologia permite a criação de aplicações web interativas utilizando a linguagem C#, sem a necessidade de JavaScript.

Também foram utilizadas as seguintes tecnologias:

.NET SDK para execução e gerenciamento do projeto
Linguagem C# para a lógica da aplicação
HTML e CSS para estrutura e estilização da interface

Heurística: Ajuda e Documentação

A aplicação desenvolvida segue a heurística de usabilidade “Ajuda e documentação”, que estabelece que, mesmo sendo um sistema fácil de usar, deve haver suporte ao usuário quando necessário.

No projeto, isso é atendido da seguinte forma:

A interface é simples e organizada, permitindo que o usuário compreenda facilmente as informações apresentadas, como perfil, experiências e habilidades.

A navegação é direta, sem elementos desnecessários, o que reduz a necessidade de instruções adicionais.

A estrutura do sistema foi organizada em partes (Models, Components e Pages), facilitando tanto a manutenção quanto a compreensão por outros desenvolvedores.

Caso seja necessário, a documentação pode ser facilmente adicionada, pois o projeto segue boas práticas de organização.

Dessa forma, o sistema garante que qualquer ajuda necessária seja fácil de acessar, clara e focada na tarefa do usuário.

Conclusão

O projeto de currículo em Blazor apresenta uma aplicação web funcional, organizada e de fácil utilização. Ele aplica conceitos importantes como separação de responsabilidades (SoC), reutilização de componentes e boas práticas de desenvolvimento.

Além disso, atende aos princípios de usabilidade, tornando a experiência do usuário simples e eficiente, sendo adequado para fins acadêmicos e demonstração de conhecimento.
